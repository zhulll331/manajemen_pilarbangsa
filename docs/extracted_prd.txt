1. Nama Produk

Pilar Digital Office

2. Deskripsi Singkat

Pilar Digital Office adalah aplikasi web dashboard internal untuk membantu pengurus inti UKM Pilar Bangsa dalam mengelola administrasi organisasi, program kerja, agenda, surat-menyurat, presensi, arsip, dan keuangan secara digital, rapi, transparan, dan minim biaya.

Aplikasi ini memiliki 3 role utama:

Ketua

Sekretaris

Bendahara

Setiap role memiliki tampilan dashboard dan fitur yang berbeda sesuai kebutuhan jabatan.

3. Tujuan Produk

Produk ini dibuat untuk:

Membantu UKM Pilar Bangsa memiliki sistem tata kelola organisasi yang modern.

Memudahkan Ketua dalam memantau kondisi organisasi secara menyeluruh.

Memudahkan Sekretaris dalam mengelola data anggota, surat, notulensi, presensi, dan arsip.

Memudahkan Bendahara dalam mencatat pemasukan, pengeluaran, iuran, dan laporan keuangan.

Mengurangi ketergantungan pada pencatatan manual atau file yang tercecer di WhatsApp.

Menjadi bukti nyata penerapan visi organisasi yang adaptif terhadap perkembangan teknologi.

4. Target Pengguna

4.1 Ketua

Ketua menggunakan aplikasi untuk melihat ringkasan organisasi, memantau program kerja, melihat agenda, membaca laporan administrasi, membaca laporan keuangan, dan memberikan evaluasi kegiatan.

4.2 Sekretaris

Sekretaris menggunakan aplikasi untuk mengelola data anggota, surat masuk, surat keluar, notulensi rapat, presensi kegiatan, dan arsip dokumen organisasi.

4.3 Bendahara

Bendahara menggunakan aplikasi untuk mengelola kas organisasi, pemasukan, pengeluaran, iuran anggota, laporan keuangan kegiatan, dan bukti transaksi.

5. Prinsip Desain

Desain aplikasi mengikuti style dashboard modern seperti education management dashboard dengan ciri-ciri:

Sidebar kiri berwarna ungu tua.

Background halaman abu muda.

Card berwarna putih dengan sudut membulat.

Shadow halus pada card.

Header atas berisi welcome text, search bar, notifikasi, dan profil user.

Menggunakan grafik visual untuk data penting.

Navigasi jelas dan mudah dipahami.

Tampilan bersih, modern, dan profesional.

Icon: Lucide React

Chart: Recharts

Database: Supabase

Hosting: Vercel

Repository: GitHub

Catatan:

Untuk tahap awal, file seperti proposal, LPJ, bukti transaksi, dan dokumen penting tidak perlu diupload langsung ke server. Cukup simpan link Google Drive agar tetap hemat biaya.

11. User Stories

Ketua

Sebagai Ketua, saya ingin melihat ringkasan organisasi agar saya bisa memantau kondisi UKM dengan cepat.

Sebagai Ketua, saya ingin melihat progress program kerja agar saya tahu kegiatan mana yang sudah selesai, berjalan, atau belum dimulai.

Sebagai Ketua, saya ingin melihat laporan keuangan ringkas agar saya bisa memastikan kondisi kas organisasi tetap sehat.

Sebagai Ketua, saya ingin memberikan evaluasi kegiatan agar pengurus bisa memperbaiki kegiatan berikutnya.

Sekretaris

Sebagai Sekretaris, saya ingin mengelola data anggota agar database UKM lebih rapi.

Sebagai Sekretaris, saya ingin mencatat surat masuk dan surat keluar agar administrasi organisasi terdokumentasi.

Sebagai Sekretaris, saya ingin menyimpan notulensi rapat agar hasil keputusan tidak hilang.

Sebagai Sekretaris, saya ingin mengelola presensi agar keaktifan anggota bisa dipantau.

Bendahara

Sebagai Bendahara, saya ingin mencatat pemasukan dan pengeluaran agar kas organisasi lebih transparan.

Sebagai Bendahara, saya ingin melihat saldo otomatis agar tidak perlu menghitung manual.

Sebagai Bendahara, saya ingin mencatat iuran anggota agar anggota yang belum lunas mudah diketahui.

Sebagai Bendahara, saya ingin menyimpan bukti transaksi agar laporan keuangan lebih valid.
